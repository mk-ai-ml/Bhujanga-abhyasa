🖼️ Figure & Axes Anatomy

Understanding the structure of matplotlib plots is essential. A figure is the overall window or canvas, and axes are the individual plots where data is drawn. The Object-Oriented (OO) interface gives you explicit control over both.

figure_axes_basics.py
import matplotlib.pyplot as plt

# Object-Oriented approach (recommended)
fig, ax = plt.subplots()
ax.plot([1, 2, 3], [1, 4, 9])
ax.set_xlabel('X Label')
ax.set_ylabel('Y Label')
ax.set_title('Plot Title')
plt.show()

# pyplot approach (quick, but less control)
plt.plot([1, 2, 3], [1, 4, 9])
plt.xlabel('X Label')
plt.ylabel('Y Label')
plt.title('Plot Title')
plt.show()
Figure & Axes Structure
Pro Tip: Use the OO Interface

The Object-Oriented interface fig, ax = plt.subplots() is more powerful and explicit. Use it for production code, especially when creating multiple subplots or saving figures.

📈 Line Plots

Line plots show trends over time or continuous data. They connect data points with lines, making patterns easy to see. You can customize markers, line styles, colors, and legends.

line_plots.py
import matplotlib.pyplot as plt

# Basic line plot
fig, ax = plt.subplots()
x = [1, 2, 3, 4, 5]
y = [2, 4, 1, 5, 3]
ax.plot(x, y, marker='o', linestyle='-', color='blue')

# Multiple lines with legend
ax.plot(x, y, label='Series A', marker='o')
ax.plot(x, [1, 3, 2, 4, 2], label='Series B', marker='s')
ax.legend()

# Different line styles
ax.plot(x, y, linestyle='-', label='Solid')
ax.plot(x, y, linestyle='--', label='Dashed')
ax.plot(x, y, linestyle=':', label='Dotted')
Marker Symbol Linestyle Pattern
'o' Circle '-' Solid line
's' Square '--' Dashed line
'^' Triangle ':' Dotted line
'*' Star '-.' Dash-dot line

📊 Bar Charts

Bar charts display categorical data as rectangular bars. Use ax.bar() for vertical bars and ax.barh() for horizontal bars. You can create grouped and stacked bar charts.

bar_charts.py
import matplotlib.pyplot as plt

# Simple bar chart
fig, ax = plt.subplots()
categories = ['A', 'B', 'C', 'D']
values = [10, 24, 36, 18]
ax.bar(categories, values, color='steelblue')

# Horizontal bars
ax.barh(categories, values)

# Grouped bars
x = range(len(categories))
width = 0.35
ax.bar([i - width/2 for i in x], values, width, label='2023')
ax.bar([i + width/2 for i in x], [14, 20, 32, 22], width, label='2024')
ax.legend()
Grouped vs. Stacked Bars

⚫ Scatter Plots

Scatter plots show the relationship between two continuous variables. Each point represents an observation. You can encode additional dimensions using color and size.

scatter_plots.py
import matplotlib.pyplot as plt
import numpy as np

# Basic scatter plot
fig, ax = plt.subplots()
x = np.random.randn(100)
y = np.random.randn(100)
ax.scatter(x, y)

# Color mapping (continuous variable)
colors = np.random.rand(100)
ax.scatter(x, y, c=colors, cmap='viridis')

# Size mapping (bubble chart)
sizes = 100 * np.random.rand(100)
ax.scatter(x, y, s=sizes, alpha=0.6)

# Combined: color and size encoding
ax.scatter(x, y, s=sizes, c=colors, cmap='plasma', alpha=0.7)

📉 Histograms & Distributions

Histograms display the distribution of a single continuous variable by binning data into ranges. You can customize bin count, add density estimates, and overlay kernel density estimates.

histograms.py
import matplotlib.pyplot as plt
import numpy as np

data = np.random.normal(100, 15, 1000)

# Basic histogram
fig, ax = plt.subplots()
ax.hist(data, bins=30)

# Normalized (density=True)
ax.hist(data, bins=30, density=True, alpha=0.7)

# Multiple distributions
ax.hist(data, bins=30, label='Data A', alpha=0.6)
ax.hist(np.random.normal(95, 10, 1000), bins=30, label='Data B', alpha=0.6)
ax.legend()
Understanding Bins

The bins parameter controls how many ranges to divide the data into. More bins = more detail but less smoothing. Start with ~30 bins and adjust based on your data.

📦 Box & Violin Plots

Box plots show quartiles, median, and outliers. Violin plots extend this by showing the full distribution shape. Both are excellent for comparing distributions across groups.

box_violin.py
import matplotlib.pyplot as plt
import numpy as np

data1 = np.random.normal(100, 15, 500)
data2 = np.random.normal(110, 20, 500)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

# Box plot
ax1.boxplot([data1, data2], labels=['Group A', 'Group B'])
ax1.set_ylabel('Value')
ax1.set_title('Box Plot')

# Violin plot
ax2.violinplot([data1, data2])
ax2.set_ylabel('Value')
ax2.set_title('Violin Plot')

🔥 Heatmaps

Heatmaps visualize 2D numerical data using colors. Use imshow() for image-like data or seaborn's heatmap() for annotated matrices with dendrograms.

heatmaps.py
import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np

# Using imshow()
data = np.random.rand(10, 10)
fig, ax = plt.subplots()
im = ax.imshow(data, cmap='coolwarm')
plt.colorbar(im)

# Using seaborn heatmap (with annotations)
corr_matrix = np.random.rand(5, 5)
sns.heatmap(corr_matrix, annot=True, fmt='.2f', cmap='viridis')

⚡ Subplots & Layouts

Combine multiple plots in one figure using subplots() for a grid layout, GridSpec for complex arrangements, or constrained_layout for automatic spacing.

subplots.py
import matplotlib.pyplot as plt

# 2x2 grid of subplots
fig, axes = plt.subplots(2, 2, figsize=(10, 8))
axes[0, 0].plot([1, 2, 3], [1, 4, 9])
axes[0, 1].scatter([1, 2, 3], [1, 4, 9])

# Shared axes
fig, axes = plt.subplots(1, 3, sharex=True, sharey=True)

# GridSpec for complex layouts
gs = fig.add_gridspec(3, 3, hspace=0.3, wspace=0.3)
ax1 = fig.add_subplot(gs[0:2, 0:2])
ax2 = fig.add_subplot(gs[0, 2])

🎨 Customization

Fine-tune every aspect of your plots: titles, labels, ticks, spines, colors, and styles. Use rcParams for global settings or set_* methods for per-plot customization.

customization.py
import matplotlib.pyplot as plt
import matplotlib as mpl

# Global rcParams
mpl.rcParams['figure.figsize'] = (12, 6)
mpl.rcParams['font.size'] = 12
mpl.rcParams['lines.linewidth'] = 2

fig, ax = plt.subplots()

# Titles and labels
ax.set_title('Chart Title', fontsize=16, fontweight='bold')
ax.set_xlabel('X Axis', fontsize=12)
ax.set_ylabel('Y Axis', fontsize=12)

# Remove spines
ax.spines['top'].set_visible(False)
ax.spines['right'].set_visible(False)

# Grid
ax.grid(True, alpha=0.3, linestyle='--')

# Tick customization
ax.tick_params(labelsize=10)
ax.set_xticks([0, 1, 2, 3])
ax.set_xticklabels(['A', 'B', 'C', 'D'])

📈 Seaborn Statistical Plots

Seaborn is built on matplotlib and provides high-level functions for statistical plots. Use it for categorical plots, relational plots, distributions, and pairwise relationships.

seaborn_plots.py
import seaborn as sns
import matplotlib.pyplot as plt

df = sns.load_dataset('iris')

# Categorical plot
sns.catplot(data=df, x='species', y='sepal_length', kind='box')

# Relational plot (scatter with hue)
sns.relplot(data=df, x='sepal_length', y='sepal_width', hue='species')

# Pairplot (all pairwise relationships)
sns.pairplot(df, hue='species')

# Distribution plot with KDE
sns.displot(df['sepal_length'], kde=True)
Seaborn Themes

Use sns.set_theme(style="darkgrid") to change the overall appearance. Available styles: darkgrid, whitegrid, dark, white, ticks.

🐼 Pandas Plot Shorthand

Pandas DataFrames have a .plot() method that creates matplotlib plots directly from your data. Use the kind parameter for different chart types.

pandas_plotting.py
import pandas as pd

df = pd.DataFrame({
    'A': [1, 2, 3, 4],
    'B': [5, 6, 7, 8]
})

# Line plot (default)
df.plot()

# Different kinds
df.plot(kind='bar')
df.plot(kind='scatter', x='A', y='B')
df.plot(kind='hist', bins=20)
df.plot(kind='box')

# With customization
df.plot(kind='bar', figsize=(10, 5), title='My Chart')
kind Parameter Plot Type Use Case
'line' Line plot Time series, trends
'bar' Bar chart Categorical comparison
'scatter' Scatter plot Correlation analysis
'hist' Histogram Distribution shape
'box' Box plot Outliers and quartiles

✅ Best Practices

Creating effective visualizations requires more than syntax. Choose the right chart type, ensure accessibility, and follow design principles.

best_practices.py
import matplotlib.pyplot as plt

# 1. Use colorblind-friendly palettes
plt.set_cmap('viridis')  # or cividis, turbo

# 2. Add meaningful titles and labels
ax.set_title('Sales Growth by Region (2024)')
ax.set_xlabel('Month')
ax.set_ylabel('Revenue ($1000s)')

# 3. Remove chart junk (unnecessary decorations)
ax.spines['top'].set_visible(False)
ax.spines['right'].set_visible(False)

# 4. Use appropriate chart types
# Time series? Use line plot
# Comparison? Use bar chart
# Distribution? Use histogram or box plot

# 5. Save figures with high DPI for print
fig.savefig('chart.png', dpi=300, bbox_inches='tight')
Chart Type Selection Guide
Accessibility Matters

Always use colorblind-friendly palettes (viridis, cividis). Don't rely on color alone to convey information. Include patterns, shapes, or labels.

Saving Figures

Use fig.savefig('file.png', dpi=300, bbox_inches='tight') for publication quality. PNG is good for screen, PDF/SVG for print.
